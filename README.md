# kaipoint
This is the source code of Kaipoint website

###Required

* Node 0.12.0 +
* MongoDb 3.0.0 +

###How to run
        
        sudo apt-get install mongodb
        npm install
        node app.js
### 开发文档

开发文档存放在 `docs/styleguide` 目录下
